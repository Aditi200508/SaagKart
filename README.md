# SaagKart
SaagKart - College project website

SaagKart is a web-based application designed to support local farmers and
women sellers by providing an online platform for selling leafy and dry
vegetables.

The system allows users to view products, register, login, place orders,
and contact the administrator. Admin can manage orders and view user
activities.

This project is developed as part of the B.Sc. Computer Applications
program under Ranchi University.

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Modules
- User Module
- Farmer Module
- Admin Module
